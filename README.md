Convierte de forma masiva archivos PNG a GIF
# Convert Images Script

Este script de Python convierte imágenes de formato PNG a GIF en un directorio especificado.

## Requisitos

Asegúrate de tener Python instalado en tu sistema. Además, 
necesitarás instalar la biblioteca Pillow para manejar las imágenes. 
Puedes instalarla usando pip:

```bash
pip install pillow

Uso
Descarga el script convert_images.py y colócalo en tu directorio de trabajo.
Modifica el script para especificar la ruta del directorio que contiene tus imágenes PNG.
Pasos para Ejecutar el Script
Edita la Ruta del Directorio:
Abre el script convert_images.py en un editor de texto o IDE y modifica la línea que especifica la ruta del directorio:
directory = r'C:\ruta\a\tu\directorio'

Ejecuta el Script:
Abre una terminal o consola, navega hasta el directorio donde está guardado convert_images.py y ejecuta el siguiente comando:
python convert_images.py
Revisa el Resultado:
El script convertirá todas las imágenes PNG en el directorio especificado a formato GIF y
las guardará en el mismo directorio con la nueva extensión.
También imprimirá un mensaje en la terminal por cada archivo convertido.
Notas
Asegúrate de que el directorio especificado contiene archivos con la extensión .png.
El script no procesa subdirectorios. Solo los archivos en el directorio especificado serán convertidos.
El script sobrescribirá los archivos GIF existentes con el mismo nombre en el directorio especificado.
